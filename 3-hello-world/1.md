# 🟢 Hello World (Go Modules)

## 1. Create Project

```bash
cd go-training
mkdir hello-world
cd hello-world
```

---

## 2. Initialize Module

```bash
go mod init github.com/krishnakumarkp/go-training/hello-world
```

---

## 3. Create `main.go`

```go
package main

import "fmt"

func main() {
	fmt.Println("Hello World!")
}
```

---

## 4. Run the Program

```bash
go run main.go
```

👉 Compiles and runs the program **without creating a binary file**

---

## 5. Build the Binary

```bash
go build
```

### 📌 What `go build` does:

* Compiles your code
* Creates an executable file in the current directory

Example:

```
hello-world   (Linux/macOS)
hello-world.exe (Windows)
```

👉 You can run it directly:

```bash
./hello-world
```

👉 Useful when:

* You want a local binary
* You’re testing builds before deployment

---

## 6. Install the Binary

```bash
go install
```

### 📌 What `go install` does:

* Compiles your program
* Installs the binary into:

```bash
$(go env GOPATH)/bin
```

Example:

```
~/go/bin/hello-world
```

👉 After this, you can run it from anywhere:

```bash
hello-world
```
