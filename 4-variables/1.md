# 🟢 Types in Go

### 🔢 Numbers

* **Integers** (`int8`, `int16`, `int32`, `int64`, `uint*`)
  → Whole numbers (signed/unsigned)
* **Floating point** (`float32`, `float64`)
  → Decimal numbers
* **Complex** (`complex64`, `complex128`)
  → Numbers with real + imaginary parts

---

### 🔤 Strings

* Sequence of characters (UTF-8 encoded)
* Immutable (cannot be changed after creation)

---

### ✅ Booleans

* `true` or `false`
* Used in conditions and control flow

---

### 🧩 Composite Types

* Built from multiple values
* Examples:

  * **Array** → fixed size collection
  * **Slice** → dynamic array (most commonly used)
  * **Struct** → group of related fields

---

### 🏷 User-Defined Types

* Create custom types from existing ones
* Improves readability and type safety

```go
type Num int
var m Num
```

---

### 🔌 Interface Types

* Define behavior (method signatures)
* Enable abstraction and decoupling

---

### 🔧 Function Types

* Functions are first-class citizens
* Can be assigned to variables, passed as arguments, returned from functions

---
