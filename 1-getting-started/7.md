# 🟢 Installing Go (Updated for 2026)

## 1. Install Go

* Download from: [https://go.dev/dl/](https://go.dev/dl/)
* Follow OS-specific instructions: [https://go.dev/doc/install](https://go.dev/doc/install)

After installation, verify:

```bash
go version
```

---

## 2. Environment Setup (Modern Go)

### ✅ No need to manually set `GOPATH` in most cases

* Go automatically uses a default:

  * Linux/macOS: `$HOME/go`
  * Windows: `%USERPROFILE%\go`

Check it with:

```bash
go env GOPATH
```

---

### ✅ Ensure Go is in PATH

#### macOS / Linux

Usually auto-configured. If not:

```bash
export PATH=$PATH:/usr/local/go/bin
export PATH=$PATH:$(go env GOPATH)/bin
```

Add to `~/.bashrc` or `~/.zshrc`.

---

#### Windows

1. Press `Windows + R` → type `sysdm.cpl`
2. Go to **Environment Variables**

Ensure:

* `Path` includes:

  ```
  C:\Program Files\Go\bin
  %USERPROFILE%\go\bin
  ```

👉 No need to set `GOPATH` manually unless you have a specific requirement.

## 3. Install VS Code

* Download: [https://code.visualstudio.com/download](https://code.visualstudio.com/download)

---

## 4. Install Go Extension

* Install: [https://marketplace.visualstudio.com/items?itemName=golang.go](https://marketplace.visualstudio.com/items?itemName=golang.go)

---

## 5. Install Go Tools in VS Code

1. Open VS Code
2. Press `Ctrl + Shift + P`
3. Run:

```
Go: Install/Update Tools
```

4. Select all tools and install

---

## 6. Verify Setup

Create a simple file:

```go
package main

import "fmt"

func main() {
    fmt.Println("Go is working!")
}
```

Run:

```bash
go run main.go
```
